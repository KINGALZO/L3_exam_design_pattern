# L3 Exam Spring Boot - Design Patterns

Ce projet est une application Spring Boot d'examen démontrant plusieurs design patterns avec une API REST simple.

## Structure du projet

- `main` : structure de base Spring Boot avec entités et services.
- `feature/factory-pattern` : implémentation du pattern Factory.
- `feature/strategy-pattern` : implémentation du pattern Strategy.
- `feature/decorator-pattern` : implémentation du pattern Decorator.
- `feature/observer-pattern` : implémentation du pattern Observer.
- `feature/api` : API REST et services exposés.

## Exécution

1. `mvn clean package`
2. `mvn spring-boot:run`

L'application s'exécute sur `http://localhost:8080`.
